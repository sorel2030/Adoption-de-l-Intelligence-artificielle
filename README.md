# Analyse de l’Adoption de l’Intelligence Artificielle dans le monde

Projet d’analyse de données explorant les tendances d’adoption de l’intelligence artificielle à l’aide de Python, du clustering et de la visualisation de données.

## Objectif
Identifier des groupes de pays ou régions présentant des comportements similaires dans l’adoption de l’intelligence artificielle.

## Technologies utilisées
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Contenu du projet
- Nettoyage et préparation des données
- Analyse exploratoire
- Clustering des données
- Visualisations des résultats
- Dashboard analytique
